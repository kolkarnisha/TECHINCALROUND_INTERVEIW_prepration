Conditional Statements in Python (The Decision-making Brain of Software) - imp for Interns

A conditional statement allows the program to execute different actions based on conditions(guardrails) -> If something is true, do this else do this

Because without conditions - software cannot:
- validate logins (mobile/laptop authentication, suggestion when battery <10 and<20)
- check eligibility (voting cards >=18, Loans, marks credit score..)
- recommend products (YT, insta recommendation based on watched category-sport,education, Devotee,shopping..)
- filter AI responses (keeping conditions in AI prompts for diff result)
- automate workflows(copilot agent mode asking permission 'allow' to do action)

Conditional statements are how backend software makes decisions

3. elif(if-elif-else) - As a real software has multiple outcomes
if / elif / else: More than two outcomes, Python checks top to bottom — FIRST True wins then Rest are skipped
Mana real systems rarely have only: yes/no decisions
Usually:
- multiple categories, multiple outcomes untaay so elif vaadali to deal with multiple conditions

Golden rule: Once a condition is True, the remaining elif/else blocks are completely ignored. Order matters — put the strictest condition first

Example: Excellent, Good, Average, Fail categories based on Marks

6. Ternary expression: (Advancement of if-else)
-> Assign one of two values based on a condition — all in one line

# Syntax raskondi
result = value_if_true  if  condition  else  value_if_false

# Real example from the project
cgpa     = 7.8
backlogs = 0

# One-line assignment cheyandiii
status  = "PASS"  if cgpa >= 6.0  else "FAIL"
clean   = "Clean" if backlogs == 0 else f"{backlogs} backlogs"
strength = "Strong" if cgpa >= 8.0 else ("Average" if cgpa >= 7.0 else "Weak")
strength = "Strong" if cgpa >= 8.0 else ("Average" if cgpa >= 7.0 and backlogs==0 else "Weak")

print(f"Status   : {status}")    # PASS
print(f"Backlogs : {clean}")     # Clean
print(f"Profile  : {strength}")  # Average

# Common in f-strings
print(f"{'Shortlisted' if cgpa >= 8.0 else 'Eligible'}")
Golden rule: Use ternary for simple label/value assignments. If the logic is complex or needs side effects (like print()), use a full if/else block instead.

# Most common mistake
# AVOID — nested ternary beyond 2 levels is unreadable
result = "A" if x>90 else "B" if x>75 else "C" if x>60 else "F"

# BETTER — use if/elif/else for 3+ outcomes
if x > 90:   result = "A"
elif x > 75: result = "B"
elif x > 60: result = "C"
else:        result = "F"
