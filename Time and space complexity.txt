Time and Space Complexity
Time Complexity?: describes how the number of operations grows as the input size grows
-> Time complexity measures how the running time of an algorithm increases as the input size n increases
-> Space complexity measures how much additional memory an algorithm requires as n increases

 O(1)       | Constant     | Accessing arr[0]            
 O(log n)  | Logarithmic  | Binary search                  
 O(n)       | Linear       | One loop through an array      
 O(n log n) | Linearithmic | Merge sort                     
 O(n²)      | Quadratic    | Two nested loops               
 O(2ⁿ)      | Exponential  | Some recursive subset problems 

##O(1) — Constant Time

for number in numbers:

#O(n²) — Quadratic Time
for i in range(len(numbers)):

        print(numbers[i], numbers[j]) # For n elements, approximately n × n operations occur

# O(log n) — Logarithmic Time - Binary search repeatedly divides the search space into half
def binary_search(numbers, target):
    left = 0
    right = len(numbers) - 1
    while left <= right:
        middle = (left + right) // 2

        if numbers[middle] == target:
            return middle
        elif numbers[middle] < target:
            left = middle + 1
        else:
            right = middle - 1
    return -1 # ikkada Time complexity: O(log n)

Example 1: One loop
def print_numbers(numbers):
    for number in numbers:
        print(number)
Ikkada Time complexity: O(n) because the loop executes n times
Space complexity: O(1) because no extra data structure grows with the input

Example 2: Nested loops
def print_pairs(numbers):
    for first in numbers:
        for second in numbers:
            print(first, second)
Time complexity: O(n²) because the inner loop runs n times for every outer-loop iteration
Space complexity: O(1)

Example 3: Creating a new list
def double_numbers(numbers):
    result = []
    for number in numbers:
        result.append(number * 2)
    return result
Time complexity: O(n)
Space complexity: O(n) because the result list stores n elements

For time complexity:
Consecutive loops are added: O(n) + O(n) = O(n)
Nested loops are multiplied: O(n) × O(n) = O(n²)
Constants are ignored: O(2n) -> O(n)
Smaller terms are ignored: O(n² + n + 5) -> O(n²)

def find_largest(numbers):
    largest = numbers[0]

    for number in numbers:
        if number > largest:
            largest = number
    return largest

numbers = [4, 8, 2, 10, 6]
print(find_largest(numbers))
##Time: O(n)
Space: O(1)

Problem 2 — Second-Largest Distinct Element
def find_second_largest(numbers):
    largest = None
    second_largest = None

    for number in numbers:
        if largest is None or number > largest:
            second_largest = largest
            largest = number
        elif number != largest and (
            second_largest is None or number > second_largest
        ):
            second_largest = number
    return second_largest
print(find_second_largest([10, 5, 8, 10, 7])) # 0
##Time: O(n)
Space: O(1)

##Problem 3 — Reverse an Array
def reverse_array(numbers):
    left = 0
    right = len(numbers) - 1

    while left < right:
        numbers[left], numbers[right] = numbers[right], numbers[left]
        left += 1
        right -= 1

    return numbers
print(reverse_array([1, 2, 3, 4, 5]))

#Complexity:
Time: O(n)
Space: O(1)

###Problem 4 — Check Whether an Array Is Sorted
def is_sorted(numbers):
    for index in range(1, len(numbers)):
        if numbers[index] < numbers[index - 1]:
            return False
    return True
print(is_sorted([1, 2, 3, 5]))
print(is_sorted([1, 4, 2, 5]))

#Complexity:
Time: O(n)

#Problem 5 — Count Even and Odd Numbers
def count_even_odd(numbers):
    even_count = 0
    odd_count = 0

    for number in numbers:
        if number % 2 == 0:
            even_count += 1
        else:
            odd_count += 1

    return even_count, odd_count
even, odd = count_even_odd([1, 2, 3, 4, 5, 6])
print("Even:", even)
print("Odd:", odd)

def calculate_sum_and_average(numbers):
    if not numbers:
        return 0, 0

    total = 0
    for number in numbers:
        total += number
    average = total / len(numbers)
    return total, average

Class and Object
class Student:
    def __init__(self, name, marks):
        self.name = name
        self.marks = marks

    def display_details(self):
        print("Name:", self.name)
        print("Marks:", self.marks)

    def has_passed(self):
        return self.marks >= 40
student1 = Student("Ravi", 85)
student1.display_details()
print(student1.has_passed())
