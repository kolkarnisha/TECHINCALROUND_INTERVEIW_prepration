4. LOOPs through CONTROL STATEMENTS

Nested Loop? - A loop inside another loop
Every patternloo ilaane untundi::
Rows -> Outer Loop
Columns -> Inner Loop

Syntax:
for row in range(...):
    for col in range(...):
# Example:
for i in range(3):
    for j in range(2):
        print(i,j)
Output:
0 0
0 1

1 0
1 1

2 0
2 1

Visual Understanding::
Outer Loop: i = 0

Inner Loop runs completely:
j = 0
j = 1

Then: i = 1
Inner Loop runs again:
j = 0
j = 1
Then: i = 2 # Inner Loop runs again

Golden Rule:: For every 1 iteration of Outer Loop, the Inner Loop executes completely

Think:
Outer Loop → Manager
Inner Loop → Employees
Elaa antee Whenever a manager is processed, all employees under him are processed

### 5-Step Formula To Solve Any Pattern
Step 1: Count Rows -> How many lines?
Step 2: Find Column Logic -> How many symbols per row?
Step 3: Identify Trend -> Increasing? / Decreasing? /Constant?
Step 4: Build Outer Loop
for row in range(...):
Step 5: Build Inner Loop
for col in range(...):
